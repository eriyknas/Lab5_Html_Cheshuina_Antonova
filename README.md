# Лабораторная работа №5

**ФИО:** Чешуина А.Д., Антонова П.А.
**Группа:** ИСП-231
**Дата:** 30.01.2026

## Описание

Мы изучили базовые теги HTML: заголовки, абзацы, списки, изображения ссылки, и т.д.

## Структура проекта

- index.html — основной HTML-файл с практическими заданиями
- index2.html — HTML-файл с практическими заданиями
- about.html — семантическая разметка
- README.md — описание лабораторной работы
- webpageCard.html - полноценная HTML-страница визитки

## Теги и элементы

1. Одиночный тег

```html
<img src="https://google.com" />
```

2. Парный тег

```html
<h1>Это заголовок</h1>
```

3. Тег абзаца <p>

```html
<p>Это абзац текста</p>
```

4. Жирный текст <strong>

```html
<p>Это <strong>очень важная</strong> информация.</p>
```

5. Курсив <em>

```html
<p>Это <em>курсивный</em> текст.</p>
```

6. Горизонтальная линия <hr>

```html
<hr />
```
